REDCap for Clinical Studies

A comprehensive course for doctoral students and clinical researchers

Designed and taught by Luca Vedovelli, Biostatistician, Clinical Trials Unit

Course Overview

This course provides comprehensive training on using REDCap (Research Electronic Data Capture) for clinical studies. Designed for doctoral students and researchers, it covers everything from basic project setup to advanced features and API integration.

Getting Started

To make the most of these materials:

  1. Review the presentation slides for conceptual understanding
  2. Follow the hands-on tutorial to practice basic REDCap functionality
  3. Try the API integration tutorial for advanced data management
  4. Explore the source code in the GitHub repository

Prerequisites

  • Basic understanding of research methodology
  • Familiarity with data management concepts
  • For API tutorials: R and RStudio installed
  • Required R packages: redcapAPI, tidyverse
  • Access to a REDCap instance
